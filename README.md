# BrainFuckValhalla
 A bf interpreter written in Odin lang
 
 ### Prerequistics:
 - odin dev-2021-12 or later
 
 ### Build the interpreter with:
 - `odin build bfv.odin`
 
 ### How to run?
 
 use the command `bfv <filename>.bf` to run the *brainF\*ck* program
